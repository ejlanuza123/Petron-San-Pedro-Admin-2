# Admin Web User Manual

Simple step-by-step guide for the deployed Admin Web Dashboard.

## 1. Before You Start

- Use a desktop browser.
- Make sure you have internet access.
- Prepare an admin email and password.

## 2. Register an Admin Account

Use this only if registration is enabled for your deployment.

1. Open the admin web app.
2. Click Register.
3. Enter your details.
4. Submit the form.
5. Wait for account confirmation.
6. Sign in after registration.

If self-registration is disabled, ask your system administrator to create your account.

## 3. Log In

1. Open the admin web app.
2. Enter your email and password.
3. Click Sign In.
4. Wait for Dashboard to load.

## 4. Dashboard (Quick Check)

Use Dashboard first every shift.

1. Check order totals and latest activity.
2. Check active deliveries.
3. Check low-stock alerts.
4. Open Orders for live dispatch actions.

## 5. Orders (Main Daily Task)

### Process a New Order

1. Open Orders.
2. Search or filter the order.
3. Open order details.
4. Assign an available rider.
5. Monitor status updates.

### Cancel an Order

1. Open the order.
2. Click cancel action.
3. Select cancellation reason.
4. Add optional note.
5. Confirm cancellation.

### Complete an Order Safely

1. Verify rider progress.
2. Confirm delivery proof is available.
3. Apply final status.
4. Recheck that order appears completed.

## 6. Products

### Add a Product

1. Open Products.
2. Click Add Product.
3. Enter name, price, stock, and category.
4. Save.

### Update a Product

1. Find the product.
2. Click Edit.
3. Update fields.
4. Save changes.

### Remove a Product

1. Find the product.
2. Click Delete.
3. Confirm deletion.

## 7. Customers

1. Open Customers.
2. Search customer name.
3. Open profile for history and details.
4. Edit profile only when necessary.

## 8. Riders

1. Open Riders.
2. Check rider availability.
3. Use Orders page to assign rider.
4. Review rider details for delays or issues.

## 9. Reports

1. Open Reports.
2. Select date range.
3. Review sales and order trends.
4. Export report if needed.

## 10. Audit Logs

1. Open Audit Logs.
2. Filter by entity type.
3. Add date range.
4. Review who changed what and when.

## 11. Settings

1. Open Settings.
2. Update value (example: default delivery fee).
3. Save setting.
4. Confirm change in live workflow.

## 12. Notifications

1. Click the notification bell.
2. Allow browser notifications when prompted.
3. Review new events regularly.

## 13. Log Out

1. Open account/profile menu.
2. Click Log Out.
3. Close browser on shared computers.

## 14. Quick Troubleshooting

### Cannot log in

1. Recheck email and password.
2. Refresh browser.
3. Confirm your admin role is active.

### Data is not updating

1. Check internet connection.
2. Refresh page.
3. Re-open affected page (Orders, Products, etc.).

### Notifications are missing

1. Check browser notification permission.
2. Keep the app tab open.
3. Refresh if realtime feed was interrupted.
